Privacy Policy

Last updated: May 4, 2026

Grab All Files ("the Extension") is a browser extension that helps users find, inspect, download, and optionally merge files from web pages they choose. This policy explains what data the Extension handles, how it is used, where it is stored, and when it is transmitted.

1. Summary of Data Handling

The Extension does not sell personal data, run third-party analytics, inject ads, use tracking scripts, or send error/crash reports to any telemetry provider in the published build. It does handle a limited amount of browsing-related data locally in your browser in order to provide its user-facing features.

This includes, for example:

  • the current tab or URL you ask the Extension to scan,
  • recent scan history stored locally for quick access, and
  • network file records (URLs and metadata of files detected via network monitoring) stored temporarily in session storage during your browsing session.

We do not send your scan history, page HTML, file contents, or browsing activity to our servers.

Trial and paid license: The Extension includes a 14-day free trial. After the trial expires, continued use requires a paid license key. Trial eligibility, license activation, license refresh, device-limit enforcement, and revocation checks are handled by the license backend using a random device UUID and, when applicable, the license key.

Access to websites: The Extension requests access to https://*/* because users can choose to scan any website and the target domain is not known in advance. This access is used to detect downloadable links and file responses on pages or URLs you choose to scan, and to download files you select. It is not used to send page HTML, browsing activity, or file contents to our servers.

2. Network Requests

The Extension makes network requests only for the following purposes:

  • License verification — When a license key is entered or revalidated, the Extension sends the license key and a randomly generated device identifier (UUID) to our server to verify the license and enforce device limits.
  • License token (JWT) issuance and refresh — After a successful verification, our server returns a short-lived signed access token (ES256 JWT, 7-day TTL) that contains the license key, device identifier, granted features, and expiry. The Extension caches this token locally so most subsequent license checks happen offline. The token is refreshed by sending the same license key and device identifier to our server, typically once a day.
  • Revocation list refresh — Once a day, the Extension downloads a small signed list of revoked token identifiers (no license keys) so revocations take effect within 24 hours, even offline. This request sends no user-identifying data.
  • Trial verification — During the free trial period, the Extension sends the device identifier (UUID) to our server to verify trial eligibility.
  • Device management — When you view or remove activated devices on the license screen, the Extension sends the license key, your current device identifier, and, if you remove a device, the target device identifier to our server.
  • User-requested page and file access — When you scan a page, enter a URL, fetch file metadata, download files, or merge PDFs, the Extension connects directly to the target website you selected. These requests do not pass through our servers.

Requests to target websites may include cookies or other session credentials already associated with that website in your browser session if they are required for the content you asked the Extension to access. The Extension does not separately collect, copy, or sell that information.

本拡張機能は、個人データの販売、サードパーティ分析、広告挿入、トラッキングスクリプトの利用を行いません。また、公開ストア配布版では、エラー報告・クラッシュ報告・テレメトリの送信も一切行いません。ユーザー向け機能を提供するため、ブラウザ内で限定的な閲覧関連データをローカル処理します。

  • スキャン対象としてユーザーが選択した現在のタブまたは入力したURL
  • クイックアクセスのために端末内へ保存される最近のスキャン履歴
  • ネットワークファイル記録（ネットワーク監視で検出されたファイルのURLとメタデータ）のセッション中の一時保存

2. ネットワークリクエスト

本拡張機能が行うネットワークリクエストは、次の目的に限られます：

  • ライセンス認証 — ライセンスキーの入力時または再検証時に、ライセンスキーとランダム生成されたデバイス識別子（UUID）を当社サーバーへ送信し、ライセンスの有効性とデバイス数制限を確認します。
  • ライセンストークン（JWT）の発行と更新 — 認証成功後、当社サーバーは短期有効な署名付きアクセストークン（ES256 JWT、有効期間7日）を返します。このトークンにはライセンスキー、デバイス識別子、付与された機能、有効期限が含まれます。本拡張機能はこのトークンを端末内にキャッシュし、その後のライセンス確認の大半をオフラインで処理します。トークンは通常1日1回、同じライセンスキーとデバイス識別子を当社サーバーへ送信して更新されます。
  • 失効リストの更新 — 1日1回、失効済みトークンID（ライセンスキーは含みません）の小さな署名付きリストをダウンロードし、オフラインでも24時間以内に失効を反映できるようにします。このリクエストにはユーザー識別情報を一切送信しません。
  • トライアル認証 — 無料トライアル期間中に、デバイス識別子（UUID）を当社サーバーへ送信し、トライアル利用資格を確認します。
  • デバイス管理 — ライセンス画面で有効化済みデバイスを表示または削除する際に、ライセンスキー、現在のデバイス識別子、および削除時には対象デバイス識別子を当社サーバーへ送信します。
  • ユーザーが指定したWebサイトへのアクセス — ページのスキャン、URL入力、ファイルメタデータ取得、ファイルダウンロード、PDF結合時に、本拡張機能はユーザーが選択した対象Webサイトへ直接接続します。これらのリクエストは当社サーバーを経由しません。

対象Webサイトへのリクエストには、そのサイトのコンテンツ取得に必要な場合、ブラウザセッションに既に紐づいているCookieやその他のセッション資格情報が含まれることがあります。本拡張機能がそれらの情報を別途収集・複製・販売することはありません。

  • Aggiornamento dell'elenco delle revoche — Una volta al giorno l'Estensione scarica un piccolo elenco firmato di identificativi di token revocati (senza chiavi di licenza) affinché le revoche abbiano effetto entro 24 ore, anche offline.   • Verifica del periodo di prova — Durante il periodo di prova gratuito, l'Estensione invia al nostro server l'identificativo del dispositivo (UUID) per verificare l'idoneità alla prova.
  • Gestione dei dispositivi — Quando Lei visualizza o rimuove i dispositivi attivati dalla schermata della licenza, l'Estensione invia al nostro server la chiave di licenza, l'identificativo del Suo dispositivo attuale e, in caso di rimozione di un dispositivo, l'identificativo del dispositivo di destinazione.

Política de Privacidade

Última atualização: 4 de maio de 2026

Grab All Files ("a Extensão") é uma extensão de navegador que ajuda os usuários a localizar, inspecionar, baixar e, opcionalmente, mesclar arquivos a partir de páginas da web por eles escolhidas. Esta política explica quais dados a Extensão manipula, como eles são usados, onde são armazenados e quando são transmitidos.

1. Resumo do Tratamento de Dados

A Extensão não vende dados pessoais, não executa análises de terceiros, não insere anúncios, não usa scripts de rastreamento e não envia relatórios de erro/falha a qualquer provedor de telemetria na versão publicada. Ela trata, sim, uma quantidade limitada de dados relacionados à navegação localmente em seu navegador para oferecer suas funcionalidades voltadas ao usuário.

Isso inclui, por exemplo:

  • a aba ou URL atual que você solicita que a Extensão escaneie,
  • o histórico recente de varreduras armazenado localmente para acesso rápido, e
  • registros de arquivos de rede (URLs e metadados de arquivos detectados via monitoramento de rede) armazenados temporariamente no armazenamento de sessão durante sua sessão de navegação.

Não enviamos seu histórico de varreduras, o HTML das páginas, o conteúdo dos arquivos ou sua atividade de navegação aos nossos servidores.

Período de avaliação e licença paga: A Extensão inclui um período de avaliação gratuita de 14 dias. Após o término do período de avaliação, o uso continuado exige uma chave de licença paga. A elegibilidade do período de avaliação, a ativação de licença, a renovação de licença, a aplicação de limite de dispositivos e as verificações de revogação são tratadas pelo backend de licenças usando um UUID aleatório do dispositivo e, quando aplicável, a chave de licença.

Acesso a sites: A Extensão solicita acesso a https://*/* porque os usuários podem optar por escanear qualquer site e o domínio de destino não é conhecido com antecedência. Esse acesso é usado para detectar links baixáveis e respostas de arquivos em páginas ou URLs que você opta por escanear, e para baixar os arquivos que você seleciona. Não é usado para enviar HTML de páginas, atividade de navegação ou conteúdo de arquivos aos nossos servidores.

2. Solicitações de Rede

A Extensão realiza solicitações de rede apenas para os seguintes fins:

  • Verificação de licença — Quando uma chave de licença é inserida ou revalidada, a Extensão envia a chave de licença e um identificador de dispositivo gerado aleatoriamente (UUID) ao nosso servidor para verificar a licença e aplicar os limites de dispositivos.
  • Emissão e renovação de token de licença (JWT) — Após uma verificação bem-sucedida, nosso servidor retorna um token de acesso assinado e de curta duração (ES256 JWT, TTL de 7 dias) que contém a chave de licença, o identificador do dispositivo, os recursos concedidos e a data de expiração. A Extensão armazena esse token localmente em cache, de forma que a maioria das verificações de licença subsequentes ocorra offline. O token é renovado pelo envio da mesma chave de licença e do identificador de dispositivo ao nosso servidor, geralmente uma vez por dia.
  • Atualização da lista de revogações — Uma vez por dia, a Extensão baixa uma pequena lista assinada de identificadores de tokens revogados (sem chaves de licença) para que as revogações entrem em vigor em até 24 horas, mesmo offline. Essa solicitação não envia nenhum dado de identificação do usuário.
  • Verificação do período de avaliação — Durante o período de avaliação gratuito, a Extensão envia o identificador do dispositivo (UUID) ao nosso servidor para verificar a elegibilidade do período de avaliação.
  • Gerenciamento de dispositivos — Ao visualizar ou remover dispositivos ativados na tela de licença, a Extensão envia ao nosso servidor a chave de licença, o identificador do seu dispositivo atual e, se você remover um dispositivo, o identificador do dispositivo de destino.
  • Acesso a páginas e arquivos solicitado pelo usuário — Ao escanear uma página, inserir uma URL, obter metadados de arquivo, baixar arquivos ou mesclar PDFs, a Extensão se conecta diretamente ao site de destino selecionado por você. Essas solicitações não passam pelos nossos servidores.

As solicitações aos sites de destino podem incluir cookies ou outras credenciais de sessão já associadas a esse site na sua sessão de navegador, caso sejam necessárias para o conteúdo que você pediu para a Extensão acessar. A Extensão não coleta, copia ou vende essas informações separadamente.
