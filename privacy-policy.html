Privacy Policy

Last updated: April 4, 2026

File Bulk Downloader ("the Extension") is a browser extension that helps users find and download files from web pages. This policy explains what data the Extension accesses, how it is used, and how it is stored.

1. Data We Collect

The Extension does not collect, transmit, or sell any personal data, browsing history, or analytics. There are no tracking scripts, cookies, or third-party analytics services.

The only network requests the Extension makes are:

  • License verification — When a license key is entered, it is sent to our server solely to verify its validity. No other data is included in this request.
  • File scanning & downloading — When you use the "Enter URL" feature, the Extension fetches web pages you specify to find downloadable files. These requests go directly to the target website; no data passes through our servers.

2. Data Stored Locally

The Extension stores the following data locally on your device using the browser's built-in storage (chrome.storage.local and chrome.storage.session). This data never leaves your device.

DataPurpose
License keyTo keep you signed in after activation
Install dateTo manage the free trial period
Download history (URLs)To mark previously downloaded files as "DL'd"
Scan historyTo show recently scanned pages for quick access
Window sizeTo remember your preferred window dimensions

You can clear all stored data at any time by uninstalling the Extension or by using your browser's extension data management.

3. Permissions

PermissionWhy it is needed
activeTabTo scan the current page for downloadable files when you click the icon
tabsTo detect tab switches for the Auto Scan feature
scriptingTo inject a content script that finds file links on the current page
downloadsTo save files to your computer
storageTo persist settings, license key, and download history locally

4. Third-Party Services

The Extension communicates with a license verification server hosted on Supabase. Only the license key string is sent; no personal information, browsing data, or device identifiers are transmitted. See Supabase's Privacy Policy for their data handling practices.

5. Children's Privacy

The Extension is not directed at children under 13. We do not knowingly collect any personal information from children.

6. Changes to This Policy

We may update this Privacy Policy from time to time. Changes will be posted on this page with an updated date. Continued use of the Extension after changes constitutes acceptance.
